# PDF MAKER
 This is an android app which converts image to pdf files.You can add image from gallery or you can scan from your phone camera.
 This project uses MVVM architechture. It creates a copy of pdf file in your downloads using iText 7 and also stores the pdf address in the room database
 it fetches data from room database and display them in recyclerview. In this recyclerview you can delete the file from your app,share it on other platform
 and display the pdf inside this app.
## LIBRARIES USED
  1. [iText 7](https://itextpdf.com/en/products/itext-7):- For pdf creation and installation in device
  2. [ScanLibrary](https://github.com/jhansireddy/AndroidScannerDemo):- For image crop and scanning
  3. [Android PdfViewer](https://github.com/barteksc/AndroidPdfViewer):- PDF Viewer # PDFMAKER
